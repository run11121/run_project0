# 🎮 Игра «Крестики-нолики»

---

### 📋 Требования к задаче
*   **Язык программирования:** Python 3.x
*   **Уровень:** Базовые знания синтаксиса (циклы, условия, списки)

---

### 📝 Описание задачи
Классическая игра **«Крестики-нолики»**, реализованная на языке Python. 

**Основные правила:**
1. Каждый игрок по очереди ставит свой знак (**X** или **O**) в пустую клетку игрового поля.
2. Победителем считается тот, кто первым соберет **3 своих знака** в линию по:
    *   ↔️ **Горизонтали**
    *   ↕️ **Вертикали**
    *   ↗️ **Диагонали**
